'''Remi''' innebär att ett [[schack]]parti slutat oavgjort.{{Schack-pos|=|tright||=|  |  |  |bd|  |  |kd|  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|pd|  |pd|  |pd|  |pd|  |=|pl|  |pl|  |pl|  |pl|  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |bl|  |  |  |=|  |  |  |  |  |  |kl|  |=|Remi: inget drag kan leda till schackmatt eftersom ingen pjäs kan passera mittlinjen på brädet.}}{{Schack-pos|=|tright||=|kl|  |  |  |  |  |  |  |=|nl|bd|kd|  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|  |  |  |  |  |  |  |  |=|Möjligt schackmatt för svart även med mycket få pjäser}}Remi är för handen om någon av följande situationer uppstår:* Spelaren vid drag (den som står i tur att göra sitt drag) kan inte utföra något drag. Detta kallas [[patt]].* Ingen av spelarna har tillräckligt med pjäser för att kunna åstadkomma [[schackmatt]], till exempel kung och löpare mot ensam kung.* Exakt samma ställning har uppkommit tre eller fler gånger och spelaren vid drag kräver remi (vilket är valfritt). * 50 drag i rad har spelats utan att en pjäs slagits eller en bonde flyttats. Detta är vanligast i slutspelsställningar.* Den ene spelaren erbjuder remi (vilket kan göras när som helst) och den andre accepterar.
